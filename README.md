# monorepo-starter

This is a monorepo starter project. It is a template for creating a monorepo project with multiple packages.

## Features

- [x] Yarn workspaces
- [x] Prettier

## Getting Started

1. Clone the repository
2. Install dependencies
3. Start the development server

```bash
git clone
cd monorepo-starter
yarn
yarn dev
```
